# AI Book Frontend

This is the frontend for the AI Robotics Textbook project, built with Docusaurus.

## Overview

The frontend is a documentation site built using Docusaurus that presents the AI & Robotics textbook content in an accessible, well-organized format.

## Features

- Interactive textbook content organized by modules
- Search functionality
- Responsive design for all devices
- Integration with the AI chatbot backend

## Tech Stack

- [Docusaurus](https://docusaurus.io/) - Static site generator
- React - Component-based UI
- TypeScript - Type safety
- MDX - Rich documentation format

## Installation

```bash
cd frontend
npm install
```

## Development

```bash
# Start development server
npm run start

# Build for production
npm run build
```

## Configuration

The site configuration is in `docusaurus.config.ts`. Key settings include:
- Site title and tagline
- Navigation structure
- Theme customization
- Deployment settings

## Deployment

This site is designed to be deployed to Vercel. The build output is in the `build` directory.
